# AXANET - Sistema de Gestión de Clientes (Migración a Python POO)

Este proyecto representa la migración y evolución del sistema de gestión de archivos de AXANET a un entorno profesional estructurado con **Python utilizando Programación Orientada a Objetos (POO)**.

## Prerrequisitos
* Python 3.9 o superior.

## Instalación y Uso
1. Crear entorno virtual: `python -m venv venv`
2. Instalar dependencias: `pip install -r requirements.txt`
3. Ejecutar aplicación: `python -m src.app`
4. Ejecutar tests: `pytest tests/ -v`
